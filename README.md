# Mohammadfahim Rathod — Portfolio

A modern, animated personal portfolio website showcasing my skills, education, and academic projects.

## Live Site

[https://rathodfahim.github.io/devfolio.github.io](https://rathodfahim.github.io/devfolio.github.io)

## Tech Stack

- **React 19** — UI library
- **Tailwind CSS 4** — Utility-first styling
- **Vite** — Lightning-fast build tool
- **Lucide React** — Icon library
- **GitHub Pages** — Hosting

## Features

- Smooth scroll-reveal animations on every section
- Typewriter effect in the hero section
- Floating contextual stickers (emojis)
- Mouse-follow glow effect
- Animated gradient background orbs
- Glassmorphism card design
- Pill-style navigation with active state indicator
- Fully responsive (mobile, tablet, desktop)
- Custom scrollbar styling

## Sections

| Section   | Description                                      |
| --------- | ------------------------------------------------ |
| Hero      | Name, role, social links, typewriter animation   |
| About     | Summary, Web Dev & Data Science highlight cards  |
| Skills    | Java, HTML/CSS, JavaScript, SQL, C++, Python     |
| Education | B.Tech IT, Diploma IT, 10th — timeline layout    |
| Projects  | Fake News Detection (AI/ML), Ping-Pong Game (JS) |
| Contact   | Email & phone with glassmorphism cards           |

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is deployed via **GitHub Pages** using GitHub Actions. Every push to `main` automatically builds and deploys.
